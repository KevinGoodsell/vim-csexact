# CSExact for Vim: Use GUI Colorschemes in the Terminal

Inspired by [CSApprox](https://www.vim.org/scripts/script.php?script_id=2390),
CSExact uses a colorscheme's GUI settings and applies them to a terminal
session. CSExact uses terminal escape codes to modify the terminal's color
palette in order to use exact RGB values.

These days you should probably enable the `termguicolors` option instead.

Please see the help file under `doc/CSExact.txt` for compatibility, limitations,
etc.

The minimum supported Vim version is 9.0.
